# E-commerce-website-using-mern-stack
E-commerce website using mern stack

The main objetif behind this projet was to build an Ecommerce website that handles both sides client and admin using the next technologies :
Mongo Db

Expressjs

Reactjs

Nodejs

Chakra Ui

Redux
